📖 Про проєкт

Цей сайт був створений насамперед для практики верстки та вдосконалення навичок роботи з CSS. Під час розробки я тренувався працювати з розташуванням елементів на сторінці, фоновими зображеннями, формами, секціями та загальною структурою лендінгу.

Також у проєкті реалізовано базову взаємодію за допомогою JavaScript для кількох кнопок і модальних вікон.
